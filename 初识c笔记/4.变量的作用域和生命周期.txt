作用域
作用域(scope)，程序设计概念，通常来说，一段程序代码中所用到的名字并不总是有效/可用的
而限定这个名字的可用性的代码范围就是这个名字的作用域

即代码在哪里起作用哪里就是它的作用域


作用域和生命周期
局部变量的作用域：就是变量所在的局部范围
全局变量的作用域：整个工程




int g_val = 2025;//全局变量

#include <stdio.h>
int main()
{
	printf("1:%d\n", g_val);
	printf("Hatsune Miku\n");
	{
		int a = 10;
		printf("2:%d\n", g_val);
		printf("a = %d\n", a);
	}
	printf("3:%d\n", g_val);


	return 0;
}


#include <stdio.h>

//声明一下变量
extern int g_val;                                使用别的项的全局变量是需要  extern  声明一下

int main()
{

	printf("%d\n", g_val);
	return 0;
}

生命周期
变量的生命周期：变量的创建和销毁之间的时间段

局部变量的生命周期：进入局部范围生命开始，出局部范围生命周期结束
全局变量的生命周期：程序的生命周期
